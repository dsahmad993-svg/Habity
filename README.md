# Habit Tracker

A fully client-side habit tracker app that works on mobile and desktop.

## Files

- index.html - app structure
- styles.css - app styling and responsive layout
- app.js - app logic, charts, storage, import/export, and encryption

## Run locally

1. Open index.html in your browser.

## Deploy on GitHub Pages

1. Push the tracker/github-ready folder contents to a GitHub repo.
2. In GitHub, open Settings -> Pages.
3. Set source to Deploy from a branch and select your branch/folder.
4. Save and wait for the site URL.

No build step is required.
